**AgentX Hackathon by Microsoft** — May–July 2025

Investigated whether LLMs solve math problems via recall or genuine reasoning, contributing to the work later accepted at **ICML 2025 NewInML Workshop** as PerplexMATH.

- Augmented Math500 (Math Olympiad) problems with 5 perturbation types: large numbers, ambiguity, impossible context, paradox, and irrelevant information
- Designed **pattern-matching algorithms** to classify reasoning steps and distinguish computation from heuristic recall
- Visualized connections between LLM response time and computation levels; wrote 1,000+ lines of Python for API calling, prompt engineering, and statistical analysis
